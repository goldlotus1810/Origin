## II. Kiến Trúc — Sinh Linh Toán Học

### 2.1 DNA — 5 Chiều Unicode

```
Sinh học:   4 mã (A, T, G, C) → ~20,000 gene → cơ thể
HomeOS:     5 mã (S, R, V, A, T) → ~5,400 biến thể → tri thức

Molecule = 5 bytes:
  [Shape]     hình dạng (8 primitives: ● ▬ ■ ▲ ○ ∪ ∩ ∖)
  [Relation]  liên kết  (8 bases: ∈ ⊂ ≡ ⊥ ∘ → ≈ ←)
  [Valence]   cảm xúc   (0x00..0xFF: tiêu cực → tích cực)
  [Arousal]   kích thích (0x00..0xFF: bình tĩnh → kích động)
  [Time]      thời gian  (Static/Slow/Medium/Fast/Instant)
```

**Đánh giá:** Thiết kế 5D orthogonal là đúng về mặt toán học. Mỗi chiều độc lập, không trùng lặp. ~5,400 ký tự Unicode có semantic identity rõ ràng — đủ làm "bộ gene" nền tảng.

**Giới hạn đã nhận diện:** LCA trung bình 2 cực → mất thông tin. Đã giải quyết bằng `lca_with_variance()`.

### 2.2 Phân tầng Crate (Dependency Graph)

```
L0 — Nền tảng (không import gì):
  ucd (412 LoC)          Unicode → Molecule lookup
```

**Đánh giá:** DAG hoàn hảo — không circular dependency. L0 không import L1 (QT14 ✅). Mỗi tầng chỉ phụ thuộc tầng dưới.

### 2.3 Neuron Analog — Mô hình thần kinh

```
DENDRITES  = STM (ngắn hạn, tự do thay đổi)
AXON       = QR (bất biến, Ed25519 signed, append-only)
SOMA       = AAM (stateless, quyết định)
SYNAPSE    = Silk edges (Hebbian: fire together → wire together)
DREAM      = Offline consolidation (STM → cluster → promote QR)
```

**Đánh giá:** Mô hình đúng về sinh học thần kinh. STM có LFU eviction (giống synapse pruning). Dream consolidation giống giấc ngủ REM. Decay φ⁻¹ ≈ 0.618 (golden ratio) là tối ưu cho tốc độ quên — đã được chứng minh trong neuroscience.
